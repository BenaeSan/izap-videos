 = Installation instructions

 * Download compressed ball and uncompress.
 * Move uncompressed folder under "mod" directory and rename to "izap-videos".
 * Go to plugin installation panel from admin panel. And browse "izap videos". Make sure it is placed below izap-elgg-bridge.
 * Click on activate. And you are done.

 > Note: It is important to register your API key at http://www.pluginlotto.com and in settings of izap-elgg-bridge. This will allow you to have access of offserver videos. If you already registered, Than you can use the existing API for that particular domain where you are installing izap-videos.
